Fundamentos do CSS

CSS é uma linguagem de estilo, também conhecida como folhas de estilo em cascata. É usada para personalização visual de um site. Ou seja, elas servem para otimizar o conteúdo das páginas e permitir uma apresentação mais amigável para o usuário.

Unidades de Medidas no CSS

Unidade de medidas absolutas:

  • cm - Centrimetros
  • mm - Milimetros
  • q - Quarto de Milimetros
  • in - Polegadas
  • pc - Paica
  • pt - Pontos
  • px - Pixels, unidade de medida absoluta mais utilizada;

Unidades de medidas relativa

Porcentagem:

  • "em" - define o tamanho da fonte(font-size) do elemento, vai utilizar como referencia o elemento pai. "1em" = 16px, "2em" = 32px, "3em" = 64px, caso voce aplique "em" sobre uma fonte de 50px, o "2em" = 100px e assim por diante. HTML, style="font-size: 1em;".
  • "rem" - tem funcionamento semelhante á unidade de medida "em"
  • viewports - é a area visivel de uma pagina web, podendo variar de dispositivos para dispositivos 1vw = 1% do tamanho da largura da area visivel da pagina. 1vh = 1% do tamanho da altura da area visivel da pagina

Dominando Grid Layouts no CSS

CSS GRID é um recurso que permite construir layouts baseados em grades bidimensionais, dividindo a página em regiões e definindo onde cada elemento deve ficar.

Nomenclatura:

grid container: elemento pai que terá a prorpiedade display:flex; especificada. Ele irá "envolver" todos os itens do grid.

grid item: elementos que são filhos diretos do grid container.

grid columns: as faixas verticais dos itens do grid, são chamadas de colunas.

grid row: as faixas horizontais dos itens do grid, são chamadas de linhas.

grid gap: espaços entre as linhas e colunas do grid.

grid line: divisorias que compõem a estrutura da grade. As estruturas entre as colunas são chamadas de column lines e, as que estão entre as faixas horizontais, são chamadas de row lines.

grid cell: são celulas, ou seja, cada "quadrado" da grade, que vai entre uma grid line e outras.

fr:unidade de medida, criada para o css grid, ela representa uma fração do espaço disponivelno container do grid.

    MEDIA QUERIES

    Estrutura : @media screen (min-width: 320px) and (max-widht: 768px)


    Media type:
  • All: Para todos os dispositivos.
  • Screen:Para monitores ou outros dispositivos com telas coloridas e com resolução adequada.
  • Handheld: Para dispositivos de mão. Normalmente com telas pequenas e banda limitada.
  • Embossed: Para dispositivos que “imprimem” em braille.
  • Tty: Para dispositivos que utilizam uma grade fixa para exibição de caracteres, como por exemplo, teletypes, terminais, dispositivos portáteis com display limitado.
  • Print: Para impressão em papel.
  • Braille: Para dispositivos táteis.
  • Projection: Para apresentações, como PowerPoint.
  • TV: Para dispositivos como televisores, ou seja, com baixa resolução, quantidade de cores e scroll limitado.
  • Speech: Para sintetizadores de voz

    • Media features

    Configura quais recursos de midia se quer combinar para aplicar determinado estilos.

  • min-width/min-height/max-width/max-height:detecta a largura e altura da viewport;
  • orientation:detecta se o dispositivo esta em modo retrato ou paisagem;
  • hover:indica que a pagina esta sendo acessada atraves de um mecanismo de ponteiro, como um mouse;
  • pointer: detecta qua preciso e o ponteiro, ajuda a detectatar o touchscreen, mouse, etc.

    • Operadores

    É possivel combinar, atraves de operadores logicos, as medias features para atender as condiçoes para aplicar os estilos.

  • AND:deve atender a todas media features.
  • OR:deve atender, no minino, uma das media features.
  • NOT:nega as condiçoes aplicadas, invertendo todo o significado do media query.
